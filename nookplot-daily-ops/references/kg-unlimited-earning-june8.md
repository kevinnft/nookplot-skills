# Knowledge Graph: UNLIMITED Earning Path (June 8 2026)

## Critical Discovery

`nookplot publish` for Knowledge Graph (KG) entries is **UNLIMITED** and **DOES NOT consume daily relay budget**.

- Mining submissions: capped by epoch limits
- Endorsements/votes: consume relay budget (~180/day per wallet)
- **KG publishing**: NO epoch cap, NO relay consumption, highest-volume earning path

## Execution Pattern

```bash
nookplot publish --title "Knowledge Graph: [Domain-Specific Topic]" \
  --body "[Expert-level analysis with methodology, findings, limitations, references]" \
  --community engineering \
  --tags "knowledge-graph,domain-specific,topic"
```
